隱私政策

最後更新日期： 2026年1月8日

歡迎使用「問星AI」（以下簡稱「本應用」）。我們非常重視您的隱私。本隱私政策旨在說明我們如何處理您的資訊。

1. 資訊收集

我們不會收集、儲存或傳輸任何您的個人身份資訊。這包括但不限於：

  • 您的姓名、電子郵件地址、電話號碼或地理位置。
  • 您輸入的任何個人資料，例如出生日期和時間。所有計算都在您的設備上本地完成，或通過匿名的 API 請求處理，我們無法將這些資訊與任何特定用戶關聯起來。

2. 數據使用

由於我們不收集任何個人數據，因此我們也不會將您的任何數據用於以下目的：

  • 個性化廣告
  • 用戶行為分析
  • 與第三方共享

您輸入的用於 AI 解讀的數據（例如星盤資訊）會被發送到我們的後端服務器（通過 Make.com 代理到 Google Gemini API）以生成解讀結果。此過程是完全匿名的，請求中不包含任何可用於識別您身份的資訊。我們和我們的服務提供商不會記錄這些請求的內容。

3. 數據儲存與安全

本應用不包含數據庫，也不會在您的設備或我們的服務器上儲存任何歷史記錄或個人資訊。您每次使用都是一次全新的、獨立的操作。

4. 第三方服務

本應用使用 Google Gemini API 提供 AI 解讀服務。我們通過一個安全的代理服務器（Make.com）來處理這些請求，以保護我們的 API 密鑰，並且不會在請求中發送任何可識別您身份的資訊。我們建議您查閱 Google 的隱私政策以了解其數據處理方式。

5. 兒童隱私

本應用不會有意收集任何來自 13 歲以下兒童的個人資訊。如果家長或監護人發現其子女在未經同意的情況下向我們提供了個人資訊，請立即與我們聯繫。

6. 政策變更

我們可能會不時更新本隱私政策。任何變更都將發佈在此頁面上。我們建議您定期查看本隱私政策以了解任何變更。
